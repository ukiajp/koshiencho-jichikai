# 次のステップ

## ここから始める

### STEP 1：GitHubリポジトリを作成する

1. https://github.com にログイン
2. 右上「+」→「New repository」
3. 以下の設定で作成：
   - Repository name：`koshiencho-jichikai`
   - Visibility：**Private**
   - 「Initialize this repository」は**チェックしない**
4. 作成後に表示される URL をコピーしておく
   （例：`https://github.com/ukiajp/koshiencho-jichikai.git`）

---
